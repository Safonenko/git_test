Виды Селекторов

  1. универсальные селекторы

    Применяется ко всем элементам , которые есть на нашей странице. Обозначается - *

    *{text-align: center;} - Весь текст на странице будет размещён по центру

  2. селекторы по названию тега

    Применяется ко всем указанным тегам

    li{color: red;} - Весь текст на странице , который находится в теге li будет отображаться красным цветом

  3. селекторы по классу

    "class" - это атрибут HTML-тега. Может быть задан любому тегу. Прописывается через - .

    - Например некому тегу b задан класс text2

    .text2{color:red;} - Текст , который находится в середине тега b будет отображаться красным цветом

  4. селекторы по id

    Применяется к индивидуальному элементу, т.е. элемент с id может быть в единственном экземпляре на странице согласно правилам семантики. Прописывается через - #

    - Например тегу h2 задан id text3

    #text3{text-align: right;} - Текст , который находится в середине тега h2 будет отображаться в правой части страницы

  5. селекторы по атрибуту

    Применяется к любому атрибуту. Прописывается через - [ ]

    - Например у нас есть тег a с атрибутом href

    a[href="http://google.com"] {color:blue;} -Текст ссылки будет отображаться синим цветом

  6. селекторы потомков

    Под CSS селекторами потомков подразумевается ситуация, когда один элемент HTML-страницы вложен в другой, причем не обязательно он должен быть первым потомком.

    - Например у нас есть некий div которму задан класс cont и нам нужно обратиться к тегу/тегам p который находится после div. Будет применено ко всем p.

    .cont p {color:red;} - Текст внутри тега/тегов p будет отображаться красным цветом.

  7. селекторы дочерние

    Дочерние селекторы CSS отличаются от селекторов потомков тем, что под дочерним подразумевается только первый потомок

    - Например у нас есть некий div которму задан класс cont после которого есть тег/теги p (только прямые потомки)

    .cont > p {text-decoration: line-through;} -Текст внутри тега/тегов p (только прямые потомки) будет перечёркнут.

  8. селекторы сестринские, или соседние (расположенные на одном уровне)

    Применяется к ближайшему указаному элементу

    - Нампример у нас есть некий тег div с заданным id="div2" и мы хотим обратиться к тегу заголовка h2 расположенным на одном уровне

    #div2 + h2 {color: red;} - Ближайший заголовок , который находится на одном уровне с div id="div2" снизу , окрасится в красный цвет.

  9. селекторы псевдоклассов (селекторы состояния)

    К селекторам псевдоклассов относятся селекторы CSS, которые описывают состояния элементов при совершении над ними какого-либо действия. Это могут быть наведение на элемент курсора мыши, клик по элементу и т.п. Записываются они через двоеточие к элементу, к которому применяются.

    - Например у нас есть некие ссылки на странице , которые имеют подчеркивание.

    a:hover {text-decoration: none;} - При наведении курсора на ссылку с эффектом подчёркивания , само подчёркивание текста исчезает.
